# ts-react
typescript + react + electron 简单起手模板

### keywords
- typescript
- react
- react-router-4
- component hot reload
- mobx
- async component
- svg icon
- 接入antd并可按需加载
- import (s)css时自动生成.(s)css.d.ts

### run
1. npm i
2. npm run dev
3. npm run launch
